# Sales Insights Dashboard

An end-to-end data analysis project built using MySQL and Power BI.

## Tools Used
- MySQL — data storage, cleaning, and querying
- Power BI — data transformation and visualization

## Project Overview
Analyzed sales transaction data across multiple Indian markets to uncover revenue trends, 
top-performing customers, and product insights. Data was cleaned and normalized from INR 
to USD in Power BI's Power Query editor before building an interactive dashboard.

## Features
- Revenue and sales quantity KPIs
- Revenue and sales volume by market
- Revenue trend over time (2017–2020)
- Top 5 customers and products by revenue
- Interactive year and month slicers
